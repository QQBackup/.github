## 关于组织

这是一群对QQ聊天记录的解密与备份感兴趣的爱好者，为了夺回本该属于所有人的数据主权，自发聚成一个兴趣小组。

秉承着开放与包容的心态，欢迎来自社区的贡献、想法与研究。

我们相信数据的价值在于可控，而可控的起点是本地备份，让每一段聊天记录回归本地，让数据的主权归于每一个拥有者。

## 关于老版本QQ

建议尽快将老版本QQ的聊天数据导入至nt_msg.db，本项目不再考虑研究msg2.0与msg3.0的数据库格式

通过官方程序的导入升级是最权威的途径，能最大限度保存数据库中的有效内容。

随着时间的发展，可用的老版数据库研究材料会越来越少，而且老版本的QQ总有一天会被服务器拒绝连接，数据库的密钥不再下发，到时候就算是神仙来了也没有办法解密数据库。
