## 关于老版本QQ

建议尽快将老版本QQ的聊天导入至nt_msg.db，本项目未来不再考虑研究msg2.0与msg3.0的数据库格式

通过官方程序的导入升级是最权威的途径，能最大限度保存数据库的内容。

随着时间的发展，可用的老版数据库研究材料会越来越少

而且老版本的QQ总有一天会被腾讯服务器拒绝连接，数据库的密钥就不会再下发了，到时候就算是神仙来了也没有办法解密数据库。
